# 🚀 Data Structures & Algorithms in Java

This repository contains my personal solutions and implementations for various Data Structures and Algorithms. I created this to track my learning progress, improve my problem-solving skills, and maintain a consistent coding habit.

## 📂 Repository Structure

The repository is organized by topic to make it easy to navigate. Each folder contains the Java solution along with comments explaining the time and space complexity.

* `src/arrays/` - Array manipulation, Two Pointers, Sliding Window
* `src/strings/` - String parsing, formatting, and matching
* `src/linkedlists/` - Singly and doubly linked list operations
* `src/trees/` - Binary Trees, BSTs, Traversals (In-progress)

## 🛠️ Tech Stack

* **Language:** Java
* **IDE:** IntelliJ IDEA

## 📝 Commit Standard

I try to keep my commit history clean and descriptive.
* `feat:` - Adding a brand new data structure or problem solution.
* `opt:` - Optimizing an existing solution (e.g., going from $O(N^2)$ to $O(N)$).
* `fix:` - Resolving a bug in the logic.
